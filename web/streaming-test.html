🔄 ストリーミング入力テスト

リアルタイムで対話的にタスクを実行できます。初期指示を入力してタスクを開始し、結果を見ながら追加の指示を送信できます。

未接続
タスクを開始すると、ここに会話が表示されます

📋 使い方

  1. 初期タスクの指示を入力して「タスク開始」をクリック
  2. Claudeの応答を確認
  3. 必要に応じて追加の指示を送信
  4. タスクが完了したら自動的に終了（または「停止」ボタンで手動終了）

🚀 テスト例

  • 段階的な分析: 「src/ディレクトリの構造を確認して」→「TypeScriptファイルをリストアップして」→「型定義に問題がないか確認して」
  • 対話的なデバッグ: 「package.jsonを確認して」→「不足している依存関係を追加して」→「テストを実行して」
  • 探索的な作業: 「このプロジェクトの概要を説明して」→「セキュリティ上の問題点を探して」→「改善案を提案して」
