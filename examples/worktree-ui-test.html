Git Worktree UI機能テスト

1. UI要素の確認

確認項目: Webインターフェースに以下の要素が表示されていることを確認してください
  • ✅ 「Git Worktreeを使用」チェックボックス
  • ✅ チェック時に表示される追加オプション（ブランチ名入力、保持オプション）
  • ✅ タスク詳細画面でWorktree使用状況の表示

2. Worktreeタスクの作成テスト

手順: ブラウザで http://localhost:5000 にアクセスして以下を実行
  1. 「実行内容」に以下を入力: 
    pwd && git branch --show-current && echo "Worktree test successful!"
  2. リポジトリを選択（cc-anywhereなど）
  3. 「Git Worktreeを使用」にチェック
  4. ブランチ名に test/ui-worktree を入力
  5. 「タスクを実行」をクリック

3. 実行結果の確認

期待される結果:
  • タスクが作成され、実行される
  • タスク詳細に「Git Worktree: ✓ 有効」と表示される
  • 指定したブランチ名が表示される
  • 実行ログに新しいWorktreeのパスが表示される（例: /.worktrees/cc-anywhere-task-xxx）
  • 実行ログに指定したブランチ名が表示される

4. Worktreeの確認

ターミナルで以下のコマンドを実行: 
# Worktreeの一覧を確認
git worktree list

# Worktreeディレクトリを確認
ls -la .worktrees/

# 作成されたブランチを確認
git branch -a | grep test/ui-worktree

5. 複数タスクの並列実行テスト

高度なテスト: 複数のWorktreeタスクを同時に実行
  1. 3つの異なるタスクを連続して作成（それぞれ異なるブランチ名を指定）
  2. 全てのタスクが独立したWorktreeで実行されることを確認
  3. git worktree list で複数のWorktreeが表示されることを確認

6. クリーンアップの確認

自動クリーンアップ: タスク完了後、設定した時間（デフォルト5分）でWorktreeが自動削除される

手動クリーンアップ:

# 特定のWorktreeを削除
git worktree remove .worktrees/cc-anywhere-task-xxx

# 全てのWorktreeをクリーンアップ
git worktree prune
